# Periodisation Model

## Marathon Phases

1. Base: 12-16 weeks
   - Build aerobic volume.
   - Keep easy running dominant.
   - Use nutrition logs to support consistency, not strict dieting.

2. Build: 8-10 weeks
   - Introduce tempo and intervals.
   - Increase long-run durability.
   - Keep carbohydrates higher around key sessions.

3. Peak: 4-6 weeks
   - Add race-specific work only when readiness gates are met.
   - Reduce lower-body strength fatigue.
   - Shrink or pause the calorie deficit if recovery or key sessions suffer.

4. Taper: 2-3 weeks
   - Reduce volume.
   - Maintain light intensity.
   - Prioritize sleep, fueling, and confidence.

## Strength Phases

- Accumulation: higher volume, technique, muscle.
- Intensification: lower reps, heavier loads.
- Maintenance: during peak running or high fatigue.

## Nutrition Across Phases

- Base: steady deficit if recovery is stable.
- Build: fuel quality sessions.
- Peak: performance and recovery beat faster weight loss.
- Taper: avoid aggressive restriction.
