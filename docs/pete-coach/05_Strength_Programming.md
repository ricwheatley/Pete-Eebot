# Strength Programming

## Model

Use 5/3/1-inspired waves where appropriate:

- Week 1: 5s
- Week 2: 3s
- Week 3: 5/3/1 or heavier top set
- Week 4: deload

This is a structure, not dogma. Readiness and running load can override it.

## Targets

- Bench: 100kg
- Squat: 100kg
- Deadlift: 140kg

## Rules

- Progress slowly.
- Maintain lower-body strength during peak run phases.
- Progress bench more readily than squat/deadlift if recovery allows.
- Do not push heavy lower body when long runs, intervals, or poor fueling make interference likely.
- Keep protein consistent; use logged nutrition as context when available.
